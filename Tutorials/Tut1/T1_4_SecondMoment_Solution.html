Tutorial 1.4  –  Second Moment of Area of a Beveled Section

Problem Statement

Calculate the second moment of area $I_x$ and $I_y$ about the x and y axes shown. Treat C as the centroid of the composite shape.

Given answer:   $I_x = 210.25 \times 10^4 \ \text{mm}^4$     $I_y = 110.25 \times 10^4 \ \text{mm}^4$

Key point: The x–y axes are assumed through the overall centroidal axes of the composite shape, C. So we do not need to calculate the centroid. We compute $I_x$ and $I_y$ directly about these axes using the parallel axis theorem applied to each part.

$$I = \bar{I} + A d^2$$

where $\bar{I}$ is the second moment about the part's own centroidal axis, and $d$ is the perpendicular distance from that centroidal axis to the reference axis.

Method 1 Three-Part Decomposition

Decompose the shape into three parts using the reference axes as natural dividers:

Fig. M1 — Three-part decomposition

The x–y axes naturally divide the shape into three parts:

  • A — base rectangle, $60 \times 30\ \text{mm}$, below x-axis
  • B — upper-right rectangle, $30 \times 50\ \text{mm}$, right of y-axis
  • C — top-left right triangle, base $30\ \text{mm}$, height $50\ \text{mm}$, left of y-axis

The dashed lines show where the axes divide the shape into its component parts.

1
Areas and Centroid Distances
Try it first, then reveal ↓
Step 1 Calculate the area and centroid distances ($d_x$, $d_y$) for each part

Calculate the area of each part and the distance from its centroid to each reference axis ($d_x$ for $I_x$; $d_y$ for $I_y$).

Figure S1 — Dimension summary for all parts

Part A — Base Rectangle (60 × 30 mm)

$$A_A = 60 \times 30 = 1{,}800\ \text{mm}^2$$

Centroid: $\bar{x}_A = 0$ (on the y-axis), $\bar{y}_A = -15\ \text{mm}$ (below x-axis)

$$d_x = |\bar{y}_A| = 15\ \text{mm} \qquad d_y = |\bar{x}_A| = 0\ \text{mm}$$

Part B — Upper-Right Rectangle (30 × 50 mm)

$$A_B = 30 \times 50 = 1{,}500\ \text{mm}^2$$

Centroid: $\bar{x}_B = +15\ \text{mm}$ (right of y-axis), $\bar{y}_B = +25\ \text{mm}$ (above x-axis)

$$d_x = |\bar{y}_B| = 25\ \text{mm} \qquad d_y = |\bar{x}_B| = 15\ \text{mm}$$

Part C — Top-Left Right Triangle (base 30 mm, height 50 mm)

The right angle sits at the origin C. The triangle centroid lies at $\tfrac{1}{3}$ of base and $\tfrac{1}{3}$ of height from the right-angle vertex:

$$A_C = \tfrac{1}{2} \times 30 \times 50 = 750\ \text{mm}^2$$
$$\bar{x}_C = -\frac{30}{3} = -10\ \text{mm} \qquad \bar{y}_C = +\frac{50}{3} \approx +16.67\ \text{mm}$$
$$d_x = |\bar{y}_C| = \frac{50}{3}\ \text{mm} \qquad d_y = |\bar{x}_C| = 10\ \text{mm}$$

Triangle centroid rule: For a right triangle with legs $b$ (horizontal) and $h$ (vertical) and the right angle at the origin, the centroid is at $\left(-\tfrac{b}{3},\ +\tfrac{h}{3}\right)$ — measured from the right-angle vertex along each leg.

2
Centroidal Second Moments $\bar{I}$
Try it first, then reveal ↓
Step 2 Apply the centroidal second moment formulas ($\bar{I}_x$, $\bar{I}_y$) for each part

The centroidal second moments $\bar{I}$ are calculated about each part's own centroidal axis before applying the parallel axis shift.

Formulas:   Rectangle: $\bar{I}_x = \dfrac{bh^3}{12}$,   $\bar{I}_y = \dfrac{hb^3}{12}$   |   Right triangle: $\bar{I}_x = \dfrac{bh^3}{36}$,   $\bar{I}_y = \dfrac{hb^3}{36}$

where $b$ is the dimension along the x-direction and $h$ is the dimension along the y-direction.

Part A — Base Rectangle ($b = 60$, $h = 30$)

$$\bar{I}_{x,A} = \frac{60 \times 30^3}{12} = \frac{60 \times 27{,}000}{12} = 135{,}000\ \text{mm}^4$$
$$\bar{I}_{y,A} = \frac{30 \times 60^3}{12} = \frac{30 \times 216{,}000}{12} = 540{,}000\ \text{mm}^4$$

Part B — Upper-Right Rectangle ($b = 30$, $h = 50$)

$$\bar{I}_{x,B} = \frac{30 \times 50^3}{12} = \frac{30 \times 125{,}000}{12} = 312{,}500\ \text{mm}^4$$
$$\bar{I}_{y,B} = \frac{50 \times 30^3}{12} = \frac{50 \times 27{,}000}{12} = 112{,}500\ \text{mm}^4$$

Part C — Right Triangle ($b = 30$, $h = 50$)

$$\bar{I}_{x,C} = \frac{30 \times 50^3}{36} = \frac{3{,}750{,}000}{36} = 104{,}167\ \text{mm}^4$$
$$\bar{I}_{y,C} = \frac{50 \times 30^3}{36} = \frac{1{,}350{,}000}{36} = 37{,}500\ \text{mm}^4$$
3
Apply Parallel Axis Theorem — $I_x$
Try it first, then reveal ↓
Step 3 Use $I_x = \bar{I}_x + A d_x^2$ for each part, then sum to get the total $I_x$

For each part:   $I_{x} = \bar{I}_x + A\,d_x^2$,   where $d_x = |\bar{y}|$ is the vertical distance from the part centroid to the x-axis.

Part A  ($d_x = 15\ \text{mm}$)

$$I_{x,A} = 135{,}000 + 1{,}800 \times 15^2 = 135{,}000 + 405{,}000 = 540{,}000\ \text{mm}^4$$

Part B  ($d_x = 25\ \text{mm}$)

$$I_{x,B} = 312{,}500 + 1{,}500 \times 25^2 = 312{,}500 + 937{,}500 = 1{,}250{,}000\ \text{mm}^4$$

Part C  ($d_x = 50/3\ \text{mm}$)

$$I_{x,C} = 104{,}167 + 750 \times \left(\frac{50}{3}\right)^2 = 104{,}167 + 750 \times \frac{2{,}500}{9} = 104{,}167 + 208{,}333 = 312{,}500\ \text{mm}^4$$

Total $I_x$

$$I_x = 540{,}000 + 1{,}250{,}000 + 312{,}500 = \mathbf{2{,}102{,}500\ \text{mm}^4}$$
4
Apply Parallel Axis Theorem — $I_y$
Try it first, then reveal ↓
Step 4 Use $I_y = \bar{I}_y + A d_y^2$ for each part, then sum to get the total $I_y$

For each part:   $I_{y} = \bar{I}_y + A\,d_y^2$,   where $d_y = |\bar{x}|$ is the horizontal distance from the part centroid to the y-axis.

Part A  ($d_y = 0$)

The centroid of Part A lies on the y-axis — no parallel axis shift needed.

$$I_{y,A} = 540{,}000 + 1{,}800 \times 0^2 = 540{,}000\ \text{mm}^4$$

Part B  ($d_y = 15\ \text{mm}$)

$$I_{y,B} = 112{,}500 + 1{,}500 \times 15^2 = 112{,}500 + 337{,}500 = 450{,}000\ \text{mm}^4$$

Part C  ($d_y = 10\ \text{mm}$)

$$I_{y,C} = 37{,}500 + 750 \times 10^2 = 37{,}500 + 75{,}000 = 112{,}500\ \text{mm}^4$$

Total $I_y$

$$I_y = 540{,}000 + 450{,}000 + 112{,}500 = \mathbf{1{,}102{,}500\ \text{mm}^4}$$

Method 2 Full Rectangle Minus Corner Triangle

Instead of three parts, treat the shape as one large bounding rectangle with a triangular corner removed.

Fig. M2 — Subtraction approach

Two shapes only:

  • ① Full bounding rectangle — $60 \times 80\ \text{mm}$, centroid at $(0,\ +10)\ \text{mm}$  (add)
  • ② Corner triangle — base $30\ \text{mm}$, height $50\ \text{mm}$, right angle at top-left corner $(-30,\ 50)$, centroid at $(-20,\ 33.3)\ \text{mm}$  (subtract)

The red region is subtracted. The dashed outline shows the full $60 \times 80\ \text{mm}$ bounding rectangle.

Note: The triangle removed in Method 2 is not the same as Part C in Method 1. They share the same dimensions ($30 \times 50$) but occupy opposite corners of the top-left region. Method 1's triangle has its right angle at C (the origin); Method 2's triangle has its right angle at the top-left corner $(-30,\ 50)$.

1
Define the Two Shapes
Try it first, then reveal ↓
Step 1 Identify the bounding rectangle and the triangle to subtract

Full bounding rectangle — $60 \times 80\ \text{mm}$ (spans $y = -30$ to $y = +50$)

$$A_\text{rect} = 60 \times 80 = 4{,}800\ \text{mm}^2 \qquad \bar{x}_\text{rect} = 0\ \text{mm}, \quad \bar{y}_\text{rect} = +10\ \text{mm}$$

Corner triangle to remove — base $30\ \text{mm}$, height $50\ \text{mm}$, right angle at $(-30,\ 50)$

$$A_\text{tri} = \tfrac{1}{2} \times 30 \times 50 = 750\ \text{mm}^2 \qquad \bar{x}_\text{tri} = -20\ \text{mm}, \quad \bar{y}_\text{tri} = +\frac{100}{3} \approx 33.33\ \text{mm}$$
2
Calculate $I_x$ by Subtraction
Try it first, then reveal ↓
Step 2 Calculate $I_x$ for the full rectangle, then subtract the triangle contribution

Full rectangle ($d_x = |\bar{y}_\text{rect}| = 10\ \text{mm}$)

$$I_{x,\text{rect}} = \frac{60 \times 80^3}{12} + 4{,}800 \times 10^2 = 2{,}560{,}000 + 480{,}000 = 3{,}040{,}000\ \text{mm}^4$$

Corner triangle ($d_x = |\bar{y}_\text{tri}| = 100/3\ \text{mm}$)

$$I_{x,\text{tri}} = \frac{30 \times 50^3}{36} + 750 \times \left(\frac{100}{3}\right)^2 = 104{,}167 + 833{,}333 = 937{,}500\ \text{mm}^4$$

Subtract

$$I_x = 3{,}040{,}000 - 937{,}500 = \mathbf{2{,}102{,}500\ \text{mm}^4}$$
3
Calculate $I_y$ by Subtraction
Try it first, then reveal ↓
Step 3 Calculate $I_y$ for the full rectangle, then subtract the triangle contribution

Full rectangle ($d_y = |\bar{x}_\text{rect}| = 0$ — centred on y-axis)

$$I_{y,\text{rect}} = \frac{80 \times 60^3}{12} + 0 = 1{,}440{,}000\ \text{mm}^4$$

Corner triangle ($d_y = |\bar{x}_\text{tri}| = 20\ \text{mm}$)

$$I_{y,\text{tri}} = \frac{50 \times 30^3}{36} + 750 \times 20^2 = 37{,}500 + 300{,}000 = 337{,}500\ \text{mm}^4$$

Subtract

$$I_y = 1{,}440{,}000 - 337{,}500 = \mathbf{1{,}102{,}500\ \text{mm}^4}$$

Both methods yield the same result:

$I_x = 210.25 \times 10^4\ \text{mm}^4$     $I_y = 110.25 \times 10^4\ \text{mm}^4$

However you slice the shape, $I_x$ and $I_y$ about the same reference axes cannot change. Checking both methods is the strongest verification in composite area problems.
